## Image Specifications

- **Max file size:** 200KB per image
- **Hero image:** 1920px wide, optimized for web
- **Service images:** 800x1000px (4:5 aspect ratio)
- **Color palette:** Warm tones, gold accents, luxury aesthetic
- **Style:** High-end, professional, editorial quality
