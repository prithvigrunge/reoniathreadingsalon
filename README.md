# Ronia Threading Salon

A luxury, high-end website for Ronia Threading Salon in San Jose, California.

## Features

- **Luxury Design System**: Gold (#C9A96E), Cream (#FAF7F2), Charcoal (#1A1A1A) color palette
- **Cinematic Video Hero**: Silent, looping background video with gradient overlays
- **Square Appointments Integration**: Embedded booking widget
- **Responsive Design**: Mobile-first, works on all devices
- **SEO Optimized**: Meta tags, sitemap, robots.txt
- **CCPA Compliant**: Cookie consent banner
- **Fast Performance**: Static export, optimized images

## Tech Stack

- Next.js 16 + React + TypeScript
- Tailwind CSS v4
- Framer Motion animations
- Square Appointments API
- Google Tag Manager

## Services

- **Threading**: Eye Brow Threading, Upper Lip, Full Face, and more
- **Waxing**: Bikini, Full Leg, Half Leg, Arms, Under Arm, and combos
- **Facial**: Herbal Facial

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Connect repo to Vercel
3. Deploy automatically

### Build Locally

```bash
npm install
npm run build
# Output in /out folder
```

## Project Structure

```
src/
├── app/              # Next.js pages
├── components/       # React components
│   ├── sections/     # Page sections
│   └── ui/           # UI components
├── config/           # Site configuration
├── data/             # Pricing & services data
├── hooks/            # Custom hooks
├── lib/              # Utilities
└── types/            # TypeScript types
```

None required - all config is in `src/config/site.ts`
